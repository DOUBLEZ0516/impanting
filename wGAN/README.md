# wGAN using Tensorflow

Tensorflow implementation of [Wasserstein GAN](https://arxiv.org/abs/1701.07875)
and [
Improved Training of Wasserstein GANs](https://arxiv.org/abs/1704.00028)

The referenced torch code can be found [here](https://github.com/Zardinality/WGAN-tensorflow).

## Prerequisites

- Python 3.6
- [Tensorflow 1.7.0](https://www.tensorflow.org/)
- [tqdm](https://github.com/tqdm/tqdm)
- [Tensorboard 1.7.0](https://www.tensorflow.org/programmers_guide/summaries_and_tensorboard)


## Usage

First, download dataset CelabA and Standford Cars.

To train a model with prepocessed dataset:

    $ python main.py
